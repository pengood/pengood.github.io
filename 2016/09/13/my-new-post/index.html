一、JSON

JSON：JavaScript 对象表示法(JavaScript Object Notation）。
JSON 是存储和交换文本信息的语法，类似 XML，JSON 比 XML 更小、更快，更易解析。
JSON建构于两种结构：

  • “键/值”对的集合，在Java中，它被理解为Map；
  • 值的有序列表，在Java中，它被理解为数组。
